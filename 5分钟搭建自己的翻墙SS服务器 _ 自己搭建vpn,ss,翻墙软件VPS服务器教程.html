5分钟搭建自己的翻墙SS服务器

    很多人都想拥有自己的VPN，或者自己的翻墙软件，其实很简单，只要你知道原理之后自己也可以搭建，本文就是教你如何在5分钟之内，搭建好自己的SS服务器（超越VPN ）的，这将是你专属的服务器，没有任何人跟你抢线，你将在谷歌的YouTube上的上与等所有网站畅游无阻！因为这套教程将同时支持windows翻墙，安卓翻墙，IOS翻墙，以及MAC和linux翻墙！  

 

    请珍惜此教程，因为，这是为数不多的亲自测试制作的，接下来，让我们开始准备遨游互联网吧！

 

1.首先是购买一个服务器，当前这家是经过数十家服务器测试后得来的，平在160毫秒左右，并且基本0掉包，美国CN2线路，10美元/月。是本人目前测试的最快速度，谷歌，YouTube上的上的秒开。并且支持支付宝付款与中文支持。力荐！

 

       购买地址>>>点击购买VPS

（进入后先选择右上角中文，然后点击注册一个账户，资料随意填写，邮箱一定要写正确）

2.选择相应的服务器，一般是Cn2 SSD L1024（注意是linux系统！），网站右上角可以切换为中文，然后选择首页的，cn2VPS,下方的linux，然后进行购买支付，跳转到管理界面。

从VPS网站首页的Cn2VPS图标点击进入即可看到如上页面。点击DEPLOY后进入购买页面如下图。一定按照提示操作！！

然后会进入到下图，勾选并选择支付方式后，直接点击完成订购付款即可。

订购成功后，在MY vps中即可看到自己的主机。

 

 

 

3.下载2个必须文件：

 

下载SS客户端

下载putty管理端

 

4.ss客户端是后期用于翻墙上网的软件，putty是我们用来连接刚才购买的vps服务器来配置ss服务的工具，解压好上面下载的2个软件，然后准备开始!!

 

5.我们首先在自己网站的后台获取IP地址，与root的密码

6.按照下图，将IP地址与密码复制出来保存在文本文档中。

7.我们现在打开putty.exe，粘贴你的IP地址，如下图。我们推荐的VPS服务器默认是22端口，所以其他不需要操作。

8.然后点击，最下方打开，这时会提示你是否信任，一定要选择“是”。于是得到如下界面。我们先输入root然后回车,然后复制自己的密码，对着窗口鼠标右键（密码是不显示的所以只要右键点一下即可），然后再按回车键。

当输入完成用户名与密码后，我们会得到下面的界面，这时候就已经连接上了自己的LinuxVPS，接下来的就非常简单了，那就是搭建我们的翻墙SS服务器！

 

9.我们首先输入下面第一行代码（复制后依然是点击鼠标右键一次粘贴），然后回车键。

yum -y install wget

然后等待运行完毕，输入第二行代码，按回车键（切记是在显示]#并出现光标的时候为运行完毕）。

wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/ssr.sh && chmod +x ssr.sh && bash ssr.sh

 

10.当上方2行代码依次执行完毕之后，我们应该会看到如下界面

11.这样我们就进入了SS的安装步骤：如果你觉得很复杂，可以配合观看>>>翻墙ss服务器搭建中文视频教程

①第一步是确认安装ShadowsocksR（SS），我们选择1。

②第二步设置端口，我习惯设置8989，记住每一步都是回车键确定。

③第三步，设置ss的密码，这里的密码可以不用很复杂根据自己喜好，这个是最后用SS客户端联网翻墙的密码，不是服务器密码。

④第四步，选择加密，我们选择10.输入10按回车即可。

⑤第五步，选择1。

⑥第六步，选择1。

⑦第七步，直接默认按回车键，意思是不限制使用设备数。

⑧第八步，直接默认按回车键，这个是单线程端口限速，我们设置为不限速。

⑨第九步，直接默认按回车键，这个是设置端口总限速，我们设置为不限速。

⑩软件需要安装相应的依赖包，这个时候，出现[y/n],输入Y然后回车即可。如下图

至此，你的SS服务器已经开始搭建环境了，静静等待自由的到来吧！

因我们用的是非常棒的服务器，所以基本不会出现掉包的情况，安装速度也是相当的快，后面你会体验到，打开SS翻墙的时候就犹如本地网络一样迅速。

 

12.当出现下面的ShadowsocksR（SS）配置信息的时候，我们就已经配置好了ss翻墙服务器了，并且是自己专属的翻墙服务器，强过VPN很多。

不过我们还需要做一个google的BBR加速处理。别担心这会非常的快速！

 

google的BBR加速，依次输入以下代码，并回车即可。

1.复制下方第一行代码，鼠标右键依次，粘贴，然后回车。

yum -y install wget

2.复制下方第二行代码，鼠标右键依次，粘贴，然后回车。

wget --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh

3.复制下方第三行代码，鼠标右键依次，粘贴，然后回车。（这是给与权限）

chmod +x bbr.sh

4.复制下方第四行代码，鼠标右键依次，粘贴，然后回车，按下任意键安装，进行最后的等待 :-)。 最后重启你的VPS即可。（点击VPS主机的网页控制面板中的reboot即可重启）

./bbr.sh

最后你应该看到如下界面

恭喜你！已经完成了SS的全部安装！你现在已经拥有自己的SS翻墙服务器了！

接下来，打开你的SS软件开始畅游互联网吧！

双击Shadowsocks.exe，在你的windows右下角有一个小飞机，双击打开，如下图填写相应的IP地址，端口，密码，选择加密方式，最后点击确定！

然后请打开google.com

OK，至此所有工作均已完成，你只需要备份你的Shadowsocks文件夹，到哪里都可以实现轻松翻墙了！哈哈，另外别忘记保存你的VPS服务器网址，因为每个月还需要续费哦！

注意！Shadowsocks打开错误请下载.net462安装重启后再测试！

 

SSR与SS的区别在于加密，如果出现有些网站自己无法访问可以尝试SSR，如果IP被封，请在VPS网上进行IP购买，几块钱很便宜，如果拿到VPS,IP就无法连接，请在网站提交工单告诉详细情况他们会处理。在使用前IP被封则是服务器运营商IP的问题。可以更换的。

当然，你可能还需要下载IOS，安卓，或者MAC的SS软件，不用担心，请点击此处，直接下载相应工具即可！因为你现在已经可以随意畅游互联网了。
